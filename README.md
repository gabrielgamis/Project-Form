# Account Registration Form

Registration form project built with **HTML, CSS, and JavaScript** to demonstrate front-end skills, responsiveness, and user interaction.

[Test my site]( https://gabrielgamis.github.io/Registration-Form/)

---

## 🔹 Features

- Fields for **first name**, **last name**, **date of birth**, **email**, **password**, and **password confirmation**  
- Basic **password validation** with show/hide toggle  
- **Gender selection** using radio buttons  
- **Responsive design** for mobile and desktop devices  
- **Interactive UI** with icons and buttons using Font Awesome  

---

## 🔹 Technologies Used

- HTML5  
- CSS3 (pure)  
- JavaScript (vanilla)  
- Font Awesome (icons)  
- Google Fonts (Poppins)
